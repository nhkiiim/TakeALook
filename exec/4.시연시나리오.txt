## 시연순서

회원가입 → **로그인** 

    → **홈페이지** = 정렬, 검색, 페이지네이션 

    → **상세페이지** = 이런상품은 어떠세요?

    → 거래 생성 

    → 20분전부터 생성가능

    → 거래 수정

    → 거래방 생성

    → 거래방 입장(나현님이 낙찰)

→ **찜** → 찜목록 → 찜취소

    1) 거래가 생성되지 않아 입장불가(멋진가방)

    2) 가격제안이 시작되어 입장불가(상춘야흥-지현님)

    3) 입장 (키보드-나현님)

→ 거래 방 입장 = 아직 거래생성 전/이미 가격제안 시작됨/입장 

→ 입장해서 buyer입장에서의 화면 채팅+ seller가 거래시작 

→ 가격제안하기 

→ 낙찰 (주영)

⇒ DM→ 메시지 보내기→ 나의 구매목록 확인 

→ seller입장에서 거래 생성 = 20분 전부터 가능한거보여주기

 → 채팅 → 가격제안 시작 → 낙찰 → DM확인

거래되지 않았고 && 예약시간이 지났 ⇒ 판매가 완료되었음으로 표시
